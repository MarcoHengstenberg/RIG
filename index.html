GridSysytem

This Grid is Percentage based. From 5 - 100%. It is seperated by 5% steps to make calculations easy and intuitive. If you want to use columns divided by 3 you can use 33% and 66%.

columns are hosted in rows and are fully responsive if you want them to.

Example:

.col.s50.m25.l20

.col.s50.m25.l20

.col.s50.m25.l20

.col.s50.m25.l20

.col.s50.m25.l20

Example:

Nesting Grids in each other

.col.s50.m50.l25

.col.s50.m50.l25

.col.s33.m50

.col.s33.l70

.col.s33

Using Tail and Offset for handling spaces between you columns
